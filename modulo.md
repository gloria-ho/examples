

## Modulo Operator

Returns the remainder of `x / y`

*Example:*

```javascript
if (num % 2 == 0) {
	console.log(`the ${num} is even`);
} else {
	console.log(`the ${num} is odd`);
}
```

*Pattern*

```javascript
5 % 5 = 0
6 % 5 = 1
7 % 5 = 2
8 % 5 = 3
9 % 5 = 4
10 % 5 = 0
11 % 5 = 1
12 % 5 = 2
13 % 5 = 3
14 % 5 = 4
15 % 5 = 0
16 % 5 = 1
```
